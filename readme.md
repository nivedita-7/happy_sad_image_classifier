# 😊 Happy vs 😢 Sad Image Classifier

A deep learning web application that classifies uploaded images as **Happy** or **Sad** using TensorFlow and Streamlit.

---

## Features

- Upload any image
- Deep Learning based classification
- Confidence score
- Clean Streamlit interface
- Modular code structure

---

## Project Structure

```text
app.py
src/
models/
notebooks/
sample_images/
```

---

## Installation

Clone repository

```bash
git clone https://github.com/username/happy-sad-classifier.git

cd happy-sad-classifier
```

Create virtual environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## Download the trained model

The model file is not included because GitHub limits files to 100 MB.

Place

```
happy_sad_classifier.keras
```

inside

```
models/
```

